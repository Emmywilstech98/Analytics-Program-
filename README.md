# Privacy Analytics

A privacy-first analytics platform that provides useful website statistics without collecting personal information.

## Vision

Privacy Analytics helps developers understand their websites while respecting visitors' privacy.

## Core Principles

- No cookies
- No fingerprinting
- No personal information
- Fast and lightweight
- Open source

## Planned Features

- Anonymous page views
- Custom events
- Live dashboard
- REST API
- JavaScript SDK
- Self-hosting support
