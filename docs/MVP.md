# Analytics Program MVP

## Goal

Allow website owners to track anonymous website analytics.

## User Journey

1. User signs up.
2. User adds a website.
3. User installs the JavaScript SDK.
4. Visitors browse the website.
5. Analytics appear in the dashboard.

## Version 1 Features

- User registration
- Login
- Website management
- JavaScript SDK
- Event collection
- Analytics dashboard

## Not Included

- Billing
- Team accounts
- AI insights
- Heatmaps
- Session replay
