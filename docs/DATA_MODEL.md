# Analytics Data Model

## Event

Each event represents one anonymous interaction.

### Fields

- Website ID
- Session ID
- Event Type
- Page URL
- Referrer
- Browser
- Operating System
- Device Type
- Screen Size
- Language
- Timestamp

## Privacy

The platform does not intentionally collect personally identifiable information (PII).

## Future Fields

- Country (privacy-preserving)
- Campaign
- Custom Events
- Conversion Goals
